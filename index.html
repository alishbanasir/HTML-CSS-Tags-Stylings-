Question no:20 Write any 5 CSS selector names along with their definitions.

Type Selector: Styles a specific element type (e.g: p, h1).

Group Selector (h1, h2): Styles multiple elements together.

Universal Selector (*): Styles all elements.

Class Selector (.classname): Styles elements with a specific class.

ID Selector (#idname): Styles an element with a specific ID.
